To start our assignment follow these steps:
1. run "npm install -g @angular/cli"
2. clone the repo
3. cd into the backend folder
4. run "npm install"
5. run "npm start"
6. cd into the db-ui folder
7. run "npm install"
8. run "ng serve"
9. open http://localhost:4200/
10. enjoy!
